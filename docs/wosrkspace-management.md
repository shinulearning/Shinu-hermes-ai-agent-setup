# Workspace Management

## Objective

Provide a centralized location for all AI-generated artifacts.

---

## Recommended Structure

```text
D:\HermesWorkspace
│
├── diagrams
├── reports
├── resumes
├── job-search
├── code
├── presentations
├── research
└── outputs
```

---

## Create Workspace

```bash
mkdir -p /mnt/d/HermesWorkspace/{diagrams,reports,resumes,job-search,code,presentations,research,outputs}
```

---

## Usage Examples

### Architecture Diagram

```text
Save output to:

/mnt/d/HermesWorkspace/diagrams/qa-platform.html
```

### Research Report

```text
Save output to:

/mnt/d/HermesWorkspace/reports/rag-study.md
```

### Code Generation

```text
Save output to:

/mnt/d/HermesWorkspace/code/playwright-project
```
